# Referencias visuales MECHA Tattoo

## Referencia principal

Instagram de Mecha:

`@mecha.ttt`

Debe utilizarse como referencia estética principal para entender:

- Estilo artístico.
- Tipo de piezas.
- Tono visual.
- Nivel de oscuridad.
- Tratamiento de imágenes.
- Universo gráfico.
- Categorías de portfolio.
- Tipo de cliente que puede conectar con la marca.

## Estética deseada

La web debe sentirse:

- Oscura.
- Gótica.
- Alternativa.
- Mágica.
- Orgánica.
- Tribal.
- Ornamental.
- Artística.
- Con energía visual.
- Premium, pero no corporativa.
- Profesional, pero no fría.
- Selectiva, no hecha para gustar a todo el mundo.

---

## Estética a evitar

Evitar completamente:

- Web comercial genérica.
- Plantilla limpia sin personalidad.
- Estética de clínica.
- Estética de marketplace.
- Colores vivos.
- Fine line.
- Lettering.
- Tatuaje comercial.
- Diseño demasiado blanco o luminoso.
- Recursos visuales típicos de tatuaje genérico.
- Calaveras, fuego, humo o clichés si no forman parte real del universo de Mecha.

---

## Dirección creativa

Claude puede proponer referencias visuales propias si encajan con el universo de Mecha, pero debe justificar por qué son coherentes.

La prioridad no es copiar una web existente, sino construir una identidad digital propia basada en:

- El branding real.
- Las fotografías disponibles.
- El estilo artístico de Mecha.
- La experiencia de usuario.
- La conversión a reservas cualificadas.

---

## Criterio de calidad

La web debe parecer el portfolio de un artista con criterio, no una página básica de servicios.

Debe tener impacto visual desde el primer segundo, pero mantener buena legibilidad, velocidad de carga y navegación clara en móvil.
