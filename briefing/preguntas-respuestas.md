## 6. Objetivo principal de la web

El objetivo principal de la web es transmitir profesionalidad, calidad artística e identidad propia.

Debe servir para posicionar a Mecha como un tatuador serio, con universo visual propio, orientado a clientes con proyectos desarrollados y también a otros estudios.

La prioridad es conseguir reservas cualificadas y mostrar una imagen profesional, artística y diferencial.

---

## 7. Dirección estratégica

La web no debe intentar gustar a todo el mundo.

Debe filtrar al público y atraer a personas que conecten con una estética oscura, alternativa, artística y con carácter.

La web debe actuar como portfolio premium y carta de presentación profesional, no como una web comercial genérica de servicios.
